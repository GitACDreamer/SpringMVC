## ```SpringMVC```

### 介绍

这是我个人在学习java web中的第一个SpringMVCDemo项目。

如果要运行这个项目，首先要创建一个"springdemo"数据库，注意选择utf-8格式，然后将springdemo.sql导入到数据库中。

### 相关环境
     - Intellij IDEA 2017.1.4 Ultimate

     - Tomcat 8.5.15

     - JDK 1.8.0_131

     - Spring 4.3.9

     - MySql 5.7.13

     - Maven 3.3.9

     - Bootstrap 3.3.5
